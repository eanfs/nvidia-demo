# 流配置示例文件
# 格式: stream_id, rtsp_url, priority, target_fps
# 
# 参数说明:
#   stream_id: 流标识符（唯一）
#   rtsp_url: RTSP 流地址
#   priority: 优先级 1-10（数字越大优先级越高）
#   target_fps: 目标检测帧率（每秒处理几帧）

# 示例配置 - 海康威视摄像头
# cam_entrance, rtsp://admin:password@192.168.1.100:554/Streaming/Channels/101, 5, 10
# cam_lobby, rtsp://admin:password@192.168.1.101:554/Streaming/Channels/101, 3, 5
# cam_parking, rtsp://admin:password@192.168.1.102:554/Streaming/Channels/101, 1, 3
